# BNNPT
Bagging Nearest-Neighbor Prediction independence Test (R Package: https://github.com/liyistat/BNNPT)


# Recommended BNNPT R package installation
```{r}
devtools::install_github("liyistat/BNNPT", force =T);
library(BNNPT);
help(bnnpt)
```
